# ☕ Seduh Coffee — Landing Page Website

Landing page **Seduh Coffee** adalah website statis yang dibuat menggunakan **HTML, CSS, dan JavaScript**, dengan desain **modern, responsif**, serta dilengkapi **gallery produk** dan **slider testimoni pelanggan**.

Project ini bertujuan untuk menampilkan profil brand kopi, memperkenalkan produk, dan meningkatkan kepercayaan pelanggan melalui tampilan visual yang menarik.

---

## ✨ Fitur Utama

* 📱 **Responsive Design** — Tampilan optimal di desktop, tablet, dan mobile
* 🖼️ **Gallery Produk** — Menampilkan foto coffee shop
* 💬 **Slider Testimoni** — Carousel testimoni pelanggan
* ⚡ **Animasi Interaktif** — Transisi halus menggunakan JavaScript
* 🎯 **Landing Page Modern** — UI clean & user-friendly

---

## 🛠️ Teknologi yang Digunakan

* **HTML5** — Struktur halaman
* **CSS3** — Styling & responsive layout (Flexbox, Media Query)
* **JavaScript (Vanilla JS)** — Interaksi, slider, dan animasi

## 🚀 Cara Menjalankan Project

1. Clone repository ini:

   ```bash
   git clone https://github.com/bayu-adjie1/seduh-coffee.git
   ```

2. Masuk ke folder project:

   ```bash
   cd seduh-coffee
   ```

3. Buka file `index.html` menggunakan browser:

   * Klik dua kali, atau
   * Gunakan **Live Server (VS Code Extension)**
